Song of the Day

Song of the day is a website application that allows you to use the Spotify API to look at different stats like the time signature to then how happy the song is. It also allows you to see your top 10 artests, and songs for a month, 6 months and lifetime! Song of the day also allows you to add one song for everyday to keep track of your mood via a song or just a new song you found that day!

Fun Holiday

Fun Holiday is a random fun holiday search. It allows you to find a fun random holiday that is happing during that day for example like "World Compliment Day." It also gives you the previus days holiday and the next days holiday!

Dinosaur

Dinosaur is a Discord Bot that is made using discord.py. It also Disecon which is a python library that I developed especially for Dinosaur as it uses a global economy system. Dinosaur also has different games like wordle and rock paper scissors! In the economy system you can buy different items from the shop that can be used to then increase your wealth in Dinosaur Points!

Disecon

Disecon is a python library made using sqlite3 that allows users to make an economy system of sorts. Disecon allows a jumping off point for a more advanced economy system built by developers!
